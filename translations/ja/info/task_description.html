あなたは英字と記号を含むテキストが与えられます。 このテキストの中で最も出現頻度の高い文字を見つけなければいけません。 文字は小文字で返さなければいけません。
最も求められている文字を検査している最中、大文字と小文字は区別しません。 つまり、検索の目的については"A" == "a"ということです。 記号、数字、空白は数えないことに注意してください。文字だけです。

もし二文字以上が同じ頻度のときは、アルファベットで最初にくる文字を返します。 例えば、"ONE"は"o", "n", "e"を各一個含んでいて、我々は"e"を選びます。

入力: 解析したいテキスト、文字列.

出力: 一番多い文字の小文字、文字列

例: 

{{init_code_tmpl}}

どのように使われるか: 多くの復号タスクでは、選択したテキストの中の様々な文字の出現頻度を知る必要があります。 例えば:文字の出現頻度を知っていれば単純な加算または減算暗号は簡単にクラックできます。 これは言語の専門家が興味をもつものです。

事前条件:
テキストはASCII記号だけを含んでいる
0 < len(text) ≤ 105
